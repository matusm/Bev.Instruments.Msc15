◆ GOMDMSC15_setDisplayMode()

int __stdcall GOMDMSC15_setDisplayMode ( int  handle,
int  mode 
)
Zu beachten
Diese Funktion ist für die folgenden Gerätetypen gültig. MSC15, MSC15-W, MSC15-Bili, CSS-45, CSS-45-WT, CSS-45-HI, CSS-D + Detector
Der Gerätetyp kann mit der Funktion GOMDMSC15_getMSC15DeviceType() ermittelt werden.

Diese Funktion aktiviert/deaktiviert bestimmte Displayfunktionen im MSC15 & CSS Gerätevariante.
Das CSS-45 besitzt kein Display, allerdings werden die Displayeinstellungen im Verbund mit einer Anzeigeeinheit (CSS-D) im CSS-45 gespeichert. Das CSS-D dient nur als reines Anzeigeelement.

Beachten Sie: Nicht alle Displayfunktionen stehen bei jeder Gerätevariante zur Verfügung. Für die verfügbaren Displayfunktionen rufen Sie die Funktion GOMDMSC15_getAvailableDisplays() auf.

Parameter
[in]handleInteger Wert > 0 zur eindeutigen Identifikation des instanziierten MSC15 & CSS Gerätevariante; dieser Wert wird von der Methode getHandle zurückgeliefert.
[in]modeInteger Wert, wird bit-weise interpretiert:
  • Bit 0: Grafik / E / CT (nur MSC15)
  • Bit 1: CRI (nur MSC15)
  • Bit 2: E / CT (nur MSC15)
  • Bit 3: ES / EP (nur MSC15)
  • Bit 4: Grafik / PAR (nur MSC15)
  • Bit 5: Grafik / Bilirubin (veraltet)
  • Bit 6: Melanopic / Brightness Factor (veraltet)
  • Bit 7: Grafik / E / CT Irradiance (nur MSC15)
  • Bit 8: frei
  • Bit 9: Radiant Power für bis zu 8 Wellenlängenbereiche (nur MSC15-W)
  • Bit 10: Irradiance für bis zu 8 Wellenlängenbereiche (nur MSC15-W)
  • Bit 11: CIE1931 Farbdreieck (nur MSC15)
  • Bit 12: Grafik / Radiant Power / Irradiance (nur MSC15-W)
  • Bit 13: Grafik / Irradiance (nur MSC15-W)
  • Bit 14: Meter Only (nur MSC15-W)
  • Bit 15: Grafik / Bilirubin IEC (nur MSC15)
  • Bit 16: Grafik / Bilirubin AAP (nur MSC15)
  • Bit 17: Melanopic (nur MSC15)
  • Bit 18: TM30-15 (nur MSC15)
  • Bit 19: Grafik / Bilirubin BLUE LED (nur MSC15-Bili)
  • Bit 20: Grafik / Bilirubin AAP(2004) (nur MSC15-Bili)
  • Bit 21: Bilirubin IEC (nur MSC15-Bili)
  • Bit 22: Bilirubin AAP(2011) (nur MSC15-Bili)
  • Bit 23: Bilirubin BLUE LED (nur MSC15-Bili)
  • Bit 24: Bilirubin AAP(2004) (nur MSC15-Bili)
Rückgabe
Rückgabewert: Integer Wert; bei Werten ungleich 0 siehe Rückgabewerttabelle.
