◆ GOMDMSC15_getTM3015()

int __stdcall GOMDMSC15_getTM3015 ( int  handle,
double *  Rf,
double *  Rg 
)

TM-30-15 is a color rendering index published by the CIE. Unlike the CRI it is based on 99 different test colors in order to obtain an accurate and adapted to all types of light rating system. The main indicators in TM-30-15, the values Rf (Fidelity Index) and Rg (Gamut Index). Rf is basically comparable to the Ra value of the CRI. The Rg value is in addition to the saturation of the colors under the measured light.

Parameters
[in]handleInteger value > 0 for unique identification of the instantiated MSC15 and CSS device type; this value is returned by the getHandle method.
[out]RfPointer to Double value: Fidelity Index
[out]RgPointer to Double value: Gammoth Index
Returns
Integer values; see return value table (warnings and errors) for values unequal to “0”.
